simdjson_warn_unused uint8_t * simdjson::fallback::dom_parser_implementation::parse_string ( const uint8_t * src,
uint8_t * dst,
bool allow_replacement ) const
finalvirtualnoexcept

Unescape a valid UTF-8 string from src to dst, stopping at a final unescaped quote. There must be an unescaped quote terminating the string. It returns the final output position as pointer. In case of error (e.g., the string has bad escaped codes), then null_nullptrptr is returned. It is assumed that the output buffer is large enough. E.g., if src points at 'joe"', then dst needs to have four free bytes + SIMDJSON_PADDING bytes.

Overridden by each implementation.

Parameters
strpointer to the beginning of a valid UTF-8 JSON string, must end with an unescaped quote.
dstpointer to a destination buffer, it must point a region in memory of sufficient size.
allow_replacementwhether we allow a replacement character when the UTF-8 contains unmatched surrogate pairs.
Returns
end of the of the written region (exclusive) or nullptr in case of error.
