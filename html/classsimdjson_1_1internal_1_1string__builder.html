Detailed Description

template<class formatter = mini_formatter>
class simdjson::internal::string_builder< formatter >

The string_builder template allows us to construct a string from a document element. It is parametrized by a "formatter" which handles the details. Thus the string_builder template could support both minification and prettification, and various other tradeoffs.
