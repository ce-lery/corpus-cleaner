Returns true if the key in this key/value pair is equal to the provided string\+\_\+view.

Design notes\+: Instead of constructing a string\+\_\+view and then comparing it with a user-\/provided strings, it is probably more performant to have dedicated functions taking as a parameter the string we want to compare against and return true when they are equal. That avoids the creation of a temporary std\+::string\+\_\+view. Though it is possible for the compiler to avoid entirely any overhead due to string\+\_\+view, relying too much on compiler magic is problematic\+: compiler magic sometimes fail, and then what do you do?
