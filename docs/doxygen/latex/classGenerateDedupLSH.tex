Calculate minhash list of text. 

Generates a sequence of hash values ​​for duplicate handling from text. ~\newline
 If two documents have the same hash value at most, the documents are considered duplicates. ~\newline
 A list of hash values, each hash value is in the format \textquotesingle{}0+07ad0b7b163f434643387f3f4799a2d466bccd0c\textquotesingle{}, ~\newline
 The first two characters represent the hash value. ~\newline
 This allows duplicate processing by pooling duplicate processing hashes into a single hash table. ~\newline
