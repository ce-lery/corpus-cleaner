\doxysubsection{Detailed Description}
The top level simdjson namespace, containing everything the library provides. 

We want to support argument-\/dependent lookup (ADL). Hence we should define operator\texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} in the namespace where the argument (here value, object, etc.) resides.
