\doxysubsection{Detailed Description}
A forward-\/only JSON array.

This is an input\+\_\+iterator, meaning\+:
\begin{DoxyItemize}
\item It is forward-\/only
\item \texorpdfstring{$\ast$}{*} must be called exactly once per element.
\item ++ must be called exactly once in between each \texorpdfstring{$\ast$}{*} (\texorpdfstring{$\ast$}{*}, ++, \texorpdfstring{$\ast$}{*}, ++, \texorpdfstring{$\ast$}{*} ...) 
\end{DoxyItemize}
