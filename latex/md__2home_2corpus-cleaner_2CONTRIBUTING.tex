I\textquotesingle{}m accepting the following issues. Please post from the issues page.


\begin{DoxyItemize}
\item Questions about documents and contents
\item Report content errors or problems
\item Suggest improvements to explanations
\item Suggest new topics etc.
\end{DoxyItemize}

Other issues are also welcome.\hypertarget{md__2home_2corpus-cleaner_2CONTRIBUTING_autotoc_md25}{}\doxysection{\texorpdfstring{Pull Request}{Pull Request}}\label{md__2home_2corpus-cleaner_2CONTRIBUTING_autotoc_md25}
Pull requests are always welcome.

I accept the following types of pull requests\+: For basic pull requests (especially detailed ones), there is no problem if you send a pull request without creating an issue.


\begin{DoxyItemize}
\item Fixed typos
\item Sample code and spelling corrections
\item Suggest or modify another explanation method
\item Improved the text to make it easier to understand
\item Website improvements
\item Test improvements
\end{DoxyItemize}

If you have any questions, such as "{}\+What about fixes/improvements like this?"{}, please raise an issue and discuss.\hypertarget{md__2home_2corpus-cleaner_2CONTRIBUTING_autotoc_md26}{}\doxysection{\texorpdfstring{How to send corrections}{How to send corrections}}\label{md__2home_2corpus-cleaner_2CONTRIBUTING_autotoc_md26}
