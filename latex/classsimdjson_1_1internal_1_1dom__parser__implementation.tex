\doxysubsection{Detailed Description}
An implementation of simdjson\textquotesingle{}s DOM parser for a particular CPU architecture.

This class is expected to be accessed only by pointer, and never move in memory (though the pointer can move).

Unescape a valid UTF-\/8 string from src to dst, stopping at a final unescaped quote. There must be an unescaped quote terminating the string. It returns the final output position as pointer. In case of error (e.\+g., the string has bad escaped codes), then null\+\_\+nullptrptr is returned. It is assumed that the output buffer is large enough. E.\+g., if src points at \textquotesingle{}joe"{}\textquotesingle{}, then dst needs to have four free bytes + SIMDJSON\+\_\+\+PADDING bytes.

Overridden by each implementation.
